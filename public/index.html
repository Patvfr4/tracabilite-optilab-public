Aide et conseils
Configuration de l'imprimante pour les bordereaux
Un mauvais paramétrage de l'imprimante est la cause la plus fréquente de bordereaux mal imprimés ou de codes-barres illisibles.
Paramètres essentiels
Format de papier : Lettre
Utiliser le format Lettre (8½ × 11 po / 216 × 279 mm) — pas le format A4 (210 × 297 mm). Le A4 est plus long et décalera tous les éléments.
Mise à l'échelle : 100 %
Ne jamais utiliser « Ajuster à la page » ou « Réduire pour s'adapter ». Ces options réduisent le document et rendent les codes-barres plus petits et parfois illisibles. Toujours sélectionner 100 % ou « Taille réelle ».
En-têtes et pieds de page : Désactivés
Les en-têtes et pieds de page ajoutés par le navigateur (URL, date, numéro de page) empiètent sur le contenu du bordereau. Les désactiver dans la boîte de dialogue d'impression.
Orientation : Portrait
Tous les formats sont générés en portrait. Le navigateur peut parfois mémoriser une orientation précédente — vérifier que Portrait est sélectionné avant d'imprimer.
Marges : Aucune ou minimales
L'application calcule ses propres marges internes. Des marges imprimante supplémentaires réduisent la zone utile. Sélectionner « Aucune » ou « Minimales » dans les paramètres de marge.
Paramètres par navigateur
Chrome & Edge
  1. Ouvrir la boîte d'impression Ctrl+P
  2. Cliquer sur « Plus de paramètres »
  3. Mise à l'échelle → 100
  4. Décocher « En-têtes et pieds de page »
  5. Marges → Aucune
  6. Vérifier que le format est Lettre
Firefox
  1. Ouvrir la boîte d'impression Ctrl+P
  2. Onglet « Mise en page »
  3. Réduire la mise à l'échelle → 100 %
  4. En-têtes/Pieds : mettre tous les champs à vide
  5. Marges → 0 sur les 4 côtés
Safari (macOS)
  1. Imprimer ⌘P
  2. Cliquer sur « Afficher les détails »
  3. Décocher « Imprimer les en-têtes et les pieds de page »
  4. Mise à l'échelle → 100 %
  5. Format de papier → US Letter
Guide par format de bordereau
Lettre pliée
Imprimer en portrait. Plier la feuille en deux dans le sens de la largeur : la partie haute devient l'étiquette d'envoi, la partie basse le bordereau d'information. À insérer dans le colis ou à agrafer.
Bordereau seul
Format standard pleine page portrait. Résumé complet avec code-barres. Convient pour les archives.
Étiquette seule
Pictogrammes IATA, adresses et départements. À coller directement sur le colis ou à insérer dans une pochette.
Grille colis
Mise en page en grille : destinataire grand format + pictogrammes + expéditeur + départements. Portrait. Destiné aux colis volumineux nécessitant une identification rapide à distance.
Pochette colis (portrait)
Dimensionné pour les enveloppes adhésives refermables 8 × 10 po (ex. ULINE S-1479). Imprimer en portrait sur papier ordinaire, glisser dans l'enveloppe plastique collée sur le colis. L'enveloppe est réutilisable — seule la feuille change à chaque envoi.
Hors-grappe (Bordereau + F-G-74)
Génère 2 pages : page 1 = bordereau de transport (étiquette IATA), page 2 = formulaire F-G-74 bilingue avec QR code et codes-barres. Les deux pages doivent être imprimées et insérées dans le colis. L'impression est obligatoire avant fermeture du modal.
Conseils qualité d'impression
Imprimante laser recommandée — Les codes-barres CODE128 et QR codes doivent être nets et contrastés. Une imprimante jet d'encre avec cartouches presque vides peut produire des codes illisibles.
Résolution minimale : 300 dpi — Vérifier dans les propriétés d'impression que la résolution est réglée à 300 dpi ou plus. Certains pilotes impriment par défaut en 150 dpi pour économiser l'encre.
Tester le scan avant déploiement — Scanner le code-barres du premier bordereau imprimé avec un lecteur de codes-barres ou l'application mobile de confirmation avant de mettre en circulation.
Sauvegarder les paramètres d'impression — Chrome et Edge mémorisent les derniers paramètres par imprimante. Configurer une fois correctement, puis cliquer sur « Imprimer » directement les prochaines fois.
Connexion et gestion de la session
Déconnexion automatique après 15 minutes d'inactivité
Si aucune interaction n'est détectée pendant 15 minutes (souris, clavier, défilement), la session se ferme automatiquement et toute saisie non enregistrée est perdue.
Formulaires longs : enregistrer régulièrement
Pour les envois avec de nombreuses listes SILP ou des notes longues, enregistrer au fil de la saisie. La session ne se prolonge pas automatiquement pendant la frappe dans certains champs.
Reconnexion automatique
Si la session expire, l'application retourne à l'écran de connexion. Se reconnecter avec le numéro d'employé et le mot de passe pour reprendre. L'historique des envois déjà enregistrés est intact.
Plusieurs onglets simultanés
L'application peut être ouverte dans plusieurs onglets avec le même compte. Éviter de modifier le même envoi dans deux onglets en même temps — le deuxième enregistrement écrasera le premier sans avertissement.
Changer de laboratoire actif
Les utilisateurs rattachés à plusieurs laboratoires peuvent basculer de contexte via le sélecteur dans la barre latérale, sans se déconnecter. Les données (envois, historique) changent automatiquement pour refléter le laboratoire sélectionné.
Mot de passe temporaire
À la première connexion avec un mot de passe fourni par un administrateur, l'application impose un changement immédiat. Choisir un mot de passe d'au moins 8 caractères. Ce changement est définitif et ne peut pas être annulé.
Confirmation de réception — envois hors-grappe
La confirmation de réception se fait par le laboratoire destinataire, sans compte dans l'application. Deux méthodes sont disponibles : en ligne (QR code / code de vérification) ou par fax.
Méthode 1 — Confirmation en ligne (recommandée)
1
Localiser le formulaire F-G-74 dans le colis
Le formulaire bilingue est imprimé et inséré dans chaque colis hors-grappe. Il contient un QR code et un code de vérification à 6 caractères (ex. A3F9B2).
2
Scanner le QR code ou accéder à la page de confirmation
Scanner le QR code avec n'importe quel appareil photo (téléphone, tablette). Ou ouvrir manuellement l'adresse imprimée sur le formulaire et saisir le N° d'envoi + le code de vérification.
3
Remplir le formulaire de confirmation
Saisir la date et l'heure de réception, le nom du réceptionnaire, et indiquer si le colis est conforme. En cas de non-conformité, sélectionner le ou les types parmi : Température · Spécimen · Emballage · Délai · Autre.
4
Soumettre
La confirmation est enregistrée instantanément. Le labo expéditeur voit le statut passer à « Reçu » (conforme) ou « Problème » (non conforme) dans ses listes en temps réel.
Méthode 2 — Confirmation par fax
Le destinataire faxe la section inférieure du F-G-74
Le destinataire complète à la main la section « À compléter par le laboratoire sous-traitant » du F-G-74 et faxe le document au numéro indiqué (selon le département : Biochimie/Hémato, Microbiologie, Pathologie ou Général).
Le superviseur saisit la confirmation dans l'application
Dans Suivi confirmations → Saisir confirmation fax, trouver l'envoi par son numéro et saisir les informations reçues par fax : date, nom, conformité.
Si aucune confirmation n'est reçue après le délai configuré (par défaut 3 jours), l'envoi passe automatiquement en statut « Aucune réponse reçue » et une alarme apparaît dans la liste du labo expéditeur.
Problèmes courants
Mon bordereau ne s'imprime pas ou la boîte de dialogue d'impression ne s'ouvre pas
Vérifier que les popups ne sont pas bloqués pour ce site. Dans Chrome/Edge : cliquer sur l'icône 🔒 dans la barre d'adresse → Autorisations du site → autoriser les popups. Firefox : une barre de notification apparaît en haut — cliquer sur « Autoriser ». L'impression utilise une fenêtre popup cachée ; si elle est bloquée, rien ne s'affiche.
Je ne vois pas un envoi dans ma liste alors qu'il vient d'être créé
L'application se met à jour en temps réel via WebSocket. Si l'envoi n'apparaît pas après quelques secondes, actualiser la page (F5). Si le problème persiste, vérifier la connexion internet — les mises à jour temps réel nécessitent une connexion active. Vérifier aussi que vous êtes sur le bon laboratoire actif (sélecteur en bas de la barre latérale).
Le code-barres sur le bordereau imprimé ne scanne pas
Causes fréquentes : (1) Mise à l'échelle ≠ 100% à l'impression — le code est trop petit ou déformé. (2) Résolution d'impression trop basse (moins de 300 dpi). (3) Cartouches d'encre presque vides — le code est trop pâle. (4) Papier brillant ou courbé qui crée des reflets. Tester en imprimant à 100%, en nettoyant la tête d'impression, ou en changeant de cartouche.
J'ai oublié de cocher un département ou j'ai sélectionné le mauvais laboratoire
Les envois peuvent être modifiés après création, tant qu'ils n'ont pas été réceptionnés. Depuis le panneau Résumé labo ou Historique, ouvrir le détail de l'envoi et cliquer sur « Modifier ». Les champs modifiables sont : transporteur, température, réfrigérant, départements et notes. Le numéro de liste et les laboratoires ne sont pas modifiables après création.
Un numéro de liste SILP est refusé avec « numéro déjà utilisé »
Chaque numéro de liste SILP est unique dans le système. Si le numéro est refusé, l'envoi existe déjà — utiliser le panneau Recherche pour le retrouver. Si la liste a physiquement été réutilisée (erreur), contacter un superviseur pour vérifier l'envoi existant et prendre la décision appropriée (déclarer perdu, réceptionner, etc.).
La page de confirmation en ligne retourne « Envoi introuvable »
Vérifications à effectuer dans l'ordre : (1) Le N° d'envoi est exact (format HG-XXXXXX-NNNNN). (2) Le code de vérification à 6 caractères est lu correctement (attention à la confusion I/1, O/0). (3) L'envoi n'a pas déjà été confirmé — un envoi déjà confirmé ne peut pas être modifié. Si tout est correct, contacter le labo expéditeur pour vérifier le statut de l'envoi.
L'application est lente ou ne répond plus
Vérifier la connexion internet en premier. L'application communique en temps réel avec Supabase — une connexion instable ralentit les chargements. Rafraîchir la page (F5) pour réinitialiser les connexions WebSocket. Si le problème persiste, contacter l'administrateur.
Je n'ai pas accès au mode Hors-grappe ou aux Bons de départ
Ces fonctionnalités sont activées par laboratoire depuis le panneau Configuration (rôle Administrateur requis). Si vous devriez y avoir accès, contacter votre administrateur pour qu'il active la fonctionnalité pour votre laboratoire dans Configuration → Hors-grappe ou Configuration → Bons de départ.
